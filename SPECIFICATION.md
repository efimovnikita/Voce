# Спецификация приложения: Mistral Speaker (PWA)

## 1. Обзор проекта
**Mistral Speaker** — это легковесное Progressive Web App (PWA), предназначенное для озвучивания текста (Text-to-Speech) с использованием Mistral AI API. Основной сценарий использования: выделение текста в браузере или любом другом приложении на смартфоне и отправка его в Mistral Speaker через стандартное меню «Поделиться».

## 2. Основные функции
*   **Интеграция с ОС (Share Target):** Появление приложения в системном списке «Поделиться» (Android/iOS).
*   **Мультиязычность:** Поддержка итальянского и английского языков.
*   **Mistral TTS API:** 
    *   Динамическое получение списка голосов через эндпоинт `/v1/audio/speech/voices`.
    *   Генерация аудио через эндпоинт `/v1/audio/speech`.
*   **Управление плеером:** Кнопки Воспроизведение/Пауза и перемотка на 10 секунд назад.
*   **Настройки:** Хранение API-ключа и ID выбранного голоса в `localStorage` браузера пользователя.
*   **Работа с контентом:** Обработка как коротких фраз, так и длинных статей без предварительного изменения текста.

## 3. Технический стек
*   **Frontend:** HTML5, CSS3 (Vanilla или Tailwind для скорости), JavaScript (ES6+).
*   **PWA:** Web App Manifest (для Share Target), Service Worker (для работы в офлайне и установки).
*   **API:** [Mistral AI TTS API](https://docs.mistral.ai/capabilities/audio/text_to_speech/speech).
*   **Хранение:** `localStorage`.

## 4. Архитектура и UI
### Экраны:
1.  **Главный экран (Плеер):** Отображает полученный текст (или его часть), статус загрузки аудио и кнопки управления (Play, Pause, -10s).
2.  **Экран настроек:** Поле ввода API-ключа и выпадающий список (Drop-down) для выбора голоса.

### Логика Share Target:
При получении текста через GET-запрос (параметр `text`), приложение автоматически:
1. Проверяет наличие API-ключа.
2. Делает запрос к Mistral для генерации аудио.
3. Начинает воспроизведение (с учетом политик автоплея браузера).

## 5. План реализации (MVP)
1.  **Настройка Манифеста:** Описание `share_target` в `manifest.json`.
2.  **Интеграция API:** Написание функции `fetchVoices()` и `generateSpeech(text)`.
3.  **Плеер:** Реализация управления через HTML5 Audio объект.
4.  **Обработка длинных текстов:** Если текст превышает лимиты Mistral (обычно это тысячи символов), добавить логику разбиения на части (chunking) для последовательного чтения.

## 6. Публикация (Бесплатный хостинг)
Самый простой способ — **Vercel**:
1.  Создать репозиторий на GitHub.
2.  Подключить его к Vercel (vercel.com).
3.  Любой "push" в ветку `main` автоматически обновляет приложение.
*Альтернатива:* **GitHub Pages** (в настройках репозитория выбрать "Static Site").
